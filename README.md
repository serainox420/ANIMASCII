> All animations come from ascii.co.ok... \
> I scrapped the living shit out of them with terminal in 20min \
> There are over 3k+ Ascii animations, each as it's separate .html file.

# More:
> Actual animation frame data is stored in each every .html, \
> In a line starting with: `n[0] =` and ending in `';`, which is scrapped & stripped from any HTML \
> It is then combined with standarized default html boilerplate \
> To open any animation file, just run it in browser, or paste file content to online sandbox like w3schools

# Extras:
## Scrapping:
1. ### Scrape EVERYTHING into one file (html) (ascii.co.uk/* -> DUMP.html)
   ```bash
   for c in {a..z} {0..9}; do curl -s "https://ascii.co.uk/animated/$c" >> DUMP.html; done
   ```
2. ### Strip URL's with .html extension into list (DUMP.html -> list.txt)
   ```bash
   cat DUMP.html | grep .html > list.txt
   ```
3. ### Cleanup the list (list.txt -> clist.txt)
   ```bash
   grep -oP '(?<=<a href=")/.*?\.html' list.txt | sed 's|^|https://ascii.co.uk/|' > clist.txt
   ```
4. ### Download HTML from all URL's in clist. (clist.txt -> .*.html)
   ```bash
   while IFS= read -r url; do
    filename=$(basename "$url")
    curl -s "$url" -o "$filename"
   done < clist.txt```
5. ### Strip every script from HTML down to frame data 
   > Frame data is stored in element: `n[0] = '<XYZ>';` where `<XYZ>` is the actual animation frame by frame in single line.
   ```bash
   sed -i '/^\s*n\[0\]/!d' *.html
   ```
6. ### Combine standardised HTML with each animation data.
   > For example preppend `prefix` to every `.html` file, then append `suffix` to every `.html` file to literally create Animated-Ascii-Sandwitch lol. Either way.
